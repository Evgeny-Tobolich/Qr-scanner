# Qr-scanner #

###### QR scanner for [Android](https://developer.android.com/)

### Scanning functions:     
* QR-codes
* Fotos
* Documents
* Bank cards
***
### Additionally:
* Auto-focus control
* Flash control
* Touch focus
* Back/Front camera
***
### Advantages
* Ease
     > Simple interface
* Comfort
     > Without advertising
* Speed
     > Fast work
